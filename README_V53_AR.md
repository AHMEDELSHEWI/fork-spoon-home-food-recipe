# شوكة ومعلقة للأكل البيتي — V53

## التعديلات الجديدة

- العميل الفضي: خصم عضوية تلقائي 5% (من 0 إلى 399 نقطة).
- العميل الذهبي: خصم عضوية تلقائي 10% (400 نقطة فأكثر).
- الخصم يطبق على قيمة المنتجات، ويُستخدم الخصم الأعلى بين العضوية وكوبون HOME5؛ لا يتم جمع الخصمين.
- عند إرسال الطلب لا تُضاف النقاط مباشرة.
- حالة الطلب تبدأ: «بانتظار قبول الدعم».
- كود قبول الطلب يغيّر الحالة فقط ولا يضيف نقاطًا.
- كود «تم الدفع» الصادر من دعم شوكة ومعلقة على 0500266720 هو الوحيد الذي يضيف النقاط، مرة واحدة فقط.
- الطلبات القديمة قبل V53 لا يعاد احتساب نقاطها.
