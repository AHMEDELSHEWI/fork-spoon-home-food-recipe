# شوكة ومعلقة للأكل البيتي — V54

تم إنشاء V54 فوق V53 مع الاحتفاظ الكامل بملفات V53 للرجوع إليها.

## التغييرات الجديدة
- بداية واضحة: طلب جديد، متابعة طلب، استفسار.
- تقسيم المنيو: محاشي، أطباق رئيسية، معجنات، حلويات، تورتات، عروض اليوم.
- استمرار مرن في التسوق وتعديل الكميات والطلب قبل التأكيد.
- اقتراحات "ممكن يعجبك كمان" داخل السلة.
- إدخال بيانات التوصيل سؤالًا بسؤال مع إمكانية تخطي الحي والشارع والبيت والعلامة المميزة.
- اختيار المدينة بين خميس مشيط وأحد رفيدة فقط.
- مشاركة اللوكيشن / Location بالموقع الحالي أو رابط Google Maps أو واتساب.
- رسوم التوصيل 20 ريال.
- خصم الفضي 5% والذهبي 10% على قيمة الأصناف فقط، وليس على رسوم التوصيل.
- ملخص نهائي قبل التأكيد مع أزرار تعديل الطلب واستمرار في التسوق.
- إرسال الطلب النهائي إلى واتساب الدعم 0500266720.

## التوافق
- ملفات V53 لم تُحذف.
- أكواد تأكيد طلبات FS53 القديمة ما زالت مدعومة.
