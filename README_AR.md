# شوكة ومعلقة للأكل البيتي — V50

هذه الحزمة تحتوي على أربعة ملفات:
1. index.html
2. .nojekyll
3. SOURCE_V50.md
4. README_AR.md

## النشر على GitHub Pages
1. فك ضغط الحزمة.
2. ارفع الملفات الأربعة إلى جذر المستودع.
3. افتح Settings ثم Pages.
4. اختر Deploy from a branch.
5. اختر main و /(root)، ثم Save.

ملاحظة: الملف `.nojekyll` ملف مخفي، لذلك قد لا يظهر في بعض تطبيقات الملفات على الهاتف إلا بعد تفعيل خيار إظهار الملفات المخفية.
