# إصلاح واجهة شوكة ومعلقة — V55

- الرجوع إلى واجهة ووظائف V53 البسيطة.
- إزالة واجهة الطلب المتسلسل من الموقع.
- إخفاء شريط السلة العائم داخل صفحة السلة وصفحة إتمام الطلب.
- تصغير شريط السلة العائم وتغيير لونه من الأسود إلى لون فاتح.
- تحسين وضوح زر المتابعة لإتمام الطلب.
- الاحتفاظ بملفات V53 وV54 كما هي للرجوع إليها.
